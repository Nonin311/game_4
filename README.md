## 中文

本项目是一个基于 Python 编写的轻量级小游戏整合程序，收录了四款经典且各具特色的文本/基础图形交互小游戏。项目由主程序进行统一管理与调度。

### 🎮 包含的游戏
1. **扫雷 (Minesweeper)**：经典的逻辑扫雷游戏，考验推理与运气。
2. **扑克牌沙蟹 (Show Hand)**：经典的扑克牌博弈游戏。
3. **五子棋 (Gomoku)**：双人对弈的经典棋类游戏。
4. **股市模拟 (Stock Market Simulator)**：简易的股票交易与市场波动模拟程序。

### 📂 项目结构
*   `game-hub.py`：项目的主入口程序（游戏大厅），负责连接和调用上述四款小游戏。

## English

This project is a lightweight Python-based game launcher that integrates four classic mini-games. All games are managed and launched through a single central interface.

### 🎮 Included Games
1. **Minesweeper**: The classic logic puzzle game of deduction and luck.
2. **Show Hand (Poker)**: A simplified classic poker-style card game.
3. **Gomoku (Five in a Row)**: A two-player abstract strategy board game.
4. **Stock Market Simulator**: A basic simulation of stock trading and price fluctuations.

### 📂 Project Structure
*   `game-hub.py`: The main entry point (Game Hub) of the application, used to launch and switch between the four games.
*   (Other corresponding `.py` source files for each game)
